# Supply Chain & Inventory Optimization Dashboard

A comprehensive Power BI dashboard designed to improve stock efficiency, reduce stockouts and overstock, and provide real-time visibility into inventory health, warehouse performance, supplier reliability, and demand forecasting.

##  Business Objective

Improve inventory efficiency by reducing stockouts and overstock while optimizing supplier performance and demand forecasting accuracy.

##  Key Insights

- **Inventory Turnover**: 12.1x with **30.2 Days Sales of Inventory (DSI)** — indicating strong stock movement
- **Stockout Rate**: 15.2% | **Overstock Rate**: 14.2%
- **Forecast Accuracy (MAPE)**: **0.10%** — Excellent forecasting performance
- **Supplier On-time Delivery**: 85.65% with a minimal **Defect Rate** of 0.10%
- **ABC Classification**: 57% of SKUs classified as A-Class, driving the majority of inventory value

##  Dashboard Pages

### 1. Inventory Health Overview
- Key KPI cards (Total SKUs, Inventory Value, Stockout %, Overstock %)
- Inventory Turnover & DSI
- ABC Classification analysis (Donut + Column Chart)

### 2. Warehouse Performance
- Inventory Value by Warehouse
- Fast-moving vs Slow-moving items
- Slow Moving Items Analysis

### 3. Supplier Analysis
- Supplier Performance KPIs
- Full Supplier Matrix
- Top 10 Suppliers (with conditional formatting)
- Lead Time Distribution

### 4. Demand Forecast
- Monthly Demand Trend vs Forecast
- Forecast vs Actual Comparison
- Forecast Accuracy (MAPE)

## 🛠 Tech Stack

- **Power BI Desktop**
- **DAX** (Data Analysis Expressions)
- **Power Query**
- Synthetic Supply Chain Dataset (Excel)

##  Dataset Information

- **Total SKUs**: 500
- **Warehouses**: 4
- **Suppliers**: 25
- **Time Period**: January 2023 – December 2025
- **Tables**: Products, Sales, Inventory_Snapshots, Purchases, Suppliers, Warehouses, Forecast

##  Features

- Interactive slicers (Warehouse, Category, ABC Class)
- Cross-page navigation buttons
- Real-time KPI cards with dynamic filtering
- Conditional formatting for performance insights
- Synced slicers across all pages
- Professional layout and color scheme

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore all four pages using the navigation buttons
4. Use slicers to filter data interactively
